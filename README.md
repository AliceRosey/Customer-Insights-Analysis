# Customer Insights Analysis

## Project Overview
This project performs exploratory data analysis (EDA) and statistical testing on a customer insights dataset to identify patterns in customer spending behavior, demographics, and engagement.

## Objectives
- Analyze customer demographics
- Study spending patterns
- Perform hypothesis testing
- Generate business insights
- Visualize customer behavior

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Dataset
The dataset contains customer demographic information, spending behavior, education level, marital status, pet ownership, and customer engagement metrics.

## Key Findings
- Gender does not significantly affect spending.
- Education level does not significantly affect spending.
- State-level spending differences are not statistically significant.
- Lifestyle factors such as marital status and pet ownership show stronger behavioral patterns.
